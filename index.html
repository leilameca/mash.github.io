Nobleza de Materiales

Bonitos por fuera, pero también hechos para aguantar

Sabemos que en exterior no solo importa que se vean bien. Por eso trabajamos con materiales pensados para sol, humedad y uso constante.

Tejido

Ratan PE Sintetico

Se ve cálido y elegante, pero está pensado para resistir mucho mejor el exterior.

  • Ayuda a conservar mejor el color frente al sol.
  • Funciona bien en ambientes con humedad o cercanos al mar.
  • Es fácil de limpiar y mantener.
  • Tiene un acabado bonito y natural a la vista.
Estructura

Acero Galvanizado

Una base firme y confiable para que el mueble se mantenga sólido con el tiempo.

  • Buena estabilidad para uso frecuente.
  • Tratamiento pensado para exterior.
  • Soporta mejor ambientes costeros.
  • Aporta durabilidad y seguridad a la pieza.
"Un mueble exterior debe verse bien el primer día, pero también seguir respondiendo bien con el uso y el tiempo."
Yascari Martínez, directora de diseño
Cuidados

Cuidados simples para que se mantengan bonitos por más tiempo

Con algunos cuidados básicos puedes conservar mejor la presencia y comodidad de tus muebles exteriores.

Estructura

Límpiala con un paño húmedo y, si hace falta, jabón neutro. Evita productos fuertes para cuidar el acabado.

Tejido PE

Puedes limpiarlo con cepillo suave, agua y shampoo delicado. Así conservas mejor su textura y brillo.

Cojinería

Si se humedece, deja secar al aire. Lava el forro en ciclo delicado y evita secadora o productos agresivos.
